1. 工具

  • github账号

注册github账号

  • node.js

让js能作为服务端脚本语言

  • npm

非官方的NodeJS包管理和分发工具

nodejs对应npm版本

  • git

分布式版本控制

  • hexo

一个快速、简洁且高效的博客框架，默认端口4000

hexo文档

2. 搭建个人仓库

  • 名字要和登录名相同【特别注意】
  • 是公开的
  • 将博客目录下的_config.yml，加入deploy设置为个人仓库名
  • 安装hexo-deployer-git自动化部署发布工具
  • 令牌
  • 发布博客

  • gitee

  • gitee pages需要实名

  • 公开且仓库名要是用户名，不需要是 用户名.gitee.io

  • 确定公钥，私人令牌

  • 部署到gitee，需要你每次推送后，去更新一下，如果想要自动更新，就需要开通 Gitee 的会员。
